# Hillway Website Improvement Roadmap

## Critical Issues (Fix Immediately)

### 1. Matt Fitzgerald Photo - 6.1MB (CRITICAL)
`images/matt-fitzgerald.jpg` is 6.1MB - larger than the entire rest of the site combined.
- Resize to max 800x800px
- Convert to WebP
- Target: under 150KB

### 2. Hidden H1 on AutoTime Page (SEO Penalty Risk)
`autotime.html:388` has a hidden H1 - Google penalizes hidden text.
- Remove or convert to meta description

### 3. Email Header Injection Risk
`contact-handler.php:263-267` - user email used directly in headers.
- Add `filter_var($email, FILTER_SANITIZE_EMAIL)`

### 4. Missing Favicons in index.html
Add to `<head>`:
```html
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
```
